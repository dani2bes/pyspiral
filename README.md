# PySpiral
Given the size of a two-dimensional array, prints a spiral using arrows

# Usage
```
uv run python main.py [size]
```
## Examples
```
1. 1x1  → 

2. 2x2  → ↓ 
        ↑ ← 
        
3. 3x3  → → ↓
        → ↓ ↓
        ↑ ← ←

4. 4x4  → → → ↓
        → → ↓ ↓
        ↑ ↑ ← ↓
        ↑ ← ← ←

5. 5x5  → → → → ↓
        → → → ↓ ↓
        ↑ → ↓ ↓ ↓
        ↑ ↑ ← ← ↓
        ↑ ← ← ← ←
```
